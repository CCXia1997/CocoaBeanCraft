# Cocoa Magic（可可魔法）mod开发记录

## 简单介绍
- 可可魔法是一个以可可豆作为科技路线的mod，通过对可可豆进行一系列处理，做成不同的巧克力，并通过巧克力一步步拓展科技树。
- 可可魔法是我目前的第一个实践类型的mod，希望能获得成长
